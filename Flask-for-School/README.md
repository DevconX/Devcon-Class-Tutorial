# Python-Flask versi sekolah, with Docker-compose

## How-to install
```bash
pip3 install Flask
```

## Or, How to use docker compose
```bash
cd docker-compose
sudo develop/up
sudo develop/shell # to get inside terminal
```

## Modules

1. Hello World
2. Static routes
3. Dynamic header from dynamic sub-route
4. Beautify (3)
5. Passing variables to HTML
6. login page
